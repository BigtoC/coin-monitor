# coin-monitor
Monitor cryptocurrency related data

## Usages
1. Monitor address native coin (ETH, MATIC, etc.) balance
2. Monitor coin price difference among exchanges

## Tech Stack
- Rust
- [ethers-rs](https://github.com/gakonst/ethers-rs)
